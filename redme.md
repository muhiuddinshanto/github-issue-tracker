var একই variable আবার declare করা যায়. এমন ভেরিয়েবল।
যার কারনে অনেক error বা bug হয়।

let => Block scoped একই variable আবার declare করা যায় না।
তবে ভেলু চেঞ্জ করা যায়।

const => Block scoped এবং value change করা যায় না, তবে নতুন কিছু এড করা যায়।


Spread operator (...) ২️⃣ Rest Parameter (...args)
সব argument ধরে এবং পুরো array আকারে দেয়।


 map()
প্রতিটি element নিয়ে কাজ করে নতুন array বানায়


২️⃣ forEach()
 প্রতিটি element নিয়ে কাজ করে, কিছু return করে না



৩️⃣ filter()
কি: element পরীক্ষা করে, condition পূরণ করা নতুন array বানায়

Arrow function হলো JavaScript এর ছোট এবং সহজ function লেখার পদ্দতি তবে সেটা ডাবল লাইন হলে return auto করে না তখন লিখে করতে হয়।

backtick ` ` ব্যবহার করা হয় string এর মধ্যে variable বসানোর জন্য।
